# cassandra-twitter
Simple usage of Cassandra driver on Java for CRUD operation

## Commands
- `register <username>`
- `follow <follower_username> <followed_username>`
- `addtweet <username> <tweet>`
- `viewtweet <username>`
- `timeline <username>`
- `exit`
